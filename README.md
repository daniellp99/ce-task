## Getting Started

- Initialize the database:

  ```bash
    npm run db:push
    # or
    yarn db:push
    # or
    pnpm db:push
    # or
    bun db:push
  ```

- First, run the development server:

  ```bash
  npm run dev
  # or
  yarn dev
  # or
  pnpm dev
  # or
  bun dev
  ```

  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
